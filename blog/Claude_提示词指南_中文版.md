# Claude 提示词指南

<!-- truncate -->

## 通用技巧

### 1. 清晰具体
- 在消息开头明确说明你的任务或问题
- 提供背景信息和细节，帮助 Claude 理解你的需求
- 将复杂任务拆解为更小的可执行步骤

**差的提示词：**
```
"帮我做个演示文稿。"
```

**好的提示词：**
```
"我需要为季度销售会议制作一个10页的PPT。内容需涵盖Q2销售表现、畅销产品以及Q3销售目标。请提供每页幻灯片的大纲和要点。"
```

**为什么更好：** 好的提示词提供了具体细节，包括幻灯片数量、会议目的以及需要涵盖的关键主题。

---

### 2. 提供示例
- 给出你期望的输出示例
- 如果你有特定的格式或风格要求，给 Claude 展示一个示例

**差的提示词：**
```
"写一封专业邮件。"
```

**好的提示词：**
```
"我需要写一封关于项目延期的专业邮件给客户。以下是我之前发过的类似邮件：

'尊敬的[客户]，
您好！关于[项目名称]，受意外问题影响，我们的完成时间将推迟约两周。我们正在努力解决，并会及时更新进展。如有疑问，请随时联系。
此致，[您的姓名]'

请参照类似的语气和结构，针对当前因供应链问题延期一个月的情况起草一封新邮件。"
```

**为什么更好：** 提供了具体的风格和语气示例，给 Claude 明确的参考点。

---

### 3. 鼓励逐步思考
- 对于复杂任务，要求 Claude "逐步思考"或"解释推理过程"
- 这能带来更准确、更详细的回答

**差的提示词：**
```
"如何提高团队生产力？"
```

**好的提示词：**
```
"我希望提高团队的生产力。请逐步思考以下因素：
1. 当前的生产力障碍（如会议过多、优先级不清晰）
2. 潜在解决方案（如时间管理技巧、项目管理工具）
3. 实施挑战
4. 效果衡量方法

每个步骤请简要说明你的推理过程，最后附上总结。"
```

**为什么更好：** 要求 Claude 系统性地思考问题，提供有结构的引导，并要求给出推理说明，同时要求最终总结便于阅读。

---

### 4. 迭代优化
- 如果 Claude 的第一次回答不够理想，要求澄清或修改
- 可以说"这个不错，但能否调整X，使其更像Y？"

**差的提示词：**
```
"改好一点。"
```

**好的提示词：**
```
"这是个好的开始，但请做以下调整：
1. 语气改得更轻松友好
2. 加入一个产品帮助客户的具体案例
3. 缩短第二段，聚焦在好处而非功能上"
```

**为什么更好：** 提供了具体的反馈和明确的改进指示，让 Claude 能进行有针对性的调整，而不是靠猜测来理解"更好"的含义。

---

### 5. 善用 Claude 的知识
- Claude 拥有跨领域的广泛知识，不要犹豫，尽管提问
- 提供相关背景和细节，让 Claude 的回答更有针对性

**差的提示词：**
```
"什么是营销？怎么做营销？"
```

**好的提示词：**
```
"我正在为一款新型环保清洁产品制定营销策略。请提供绿色营销当前趋势的概览，包括：
1. 能引起环保消费者共鸣的关键信息策略
2. 触达该受众的有效渠道
3. 过去一年成功的绿色营销活动案例
4. 需要避免的潜在陷阱（如被指控洗绿）

这些信息将帮助我制定营销方向。"
```

**为什么更好：** 要求具体且与情境相关的信息，提供了使用背景，帮助 Claude 以最相关的方式组织回答。

---

### 6. 使用角色扮演
- 让 Claude 扮演特定角色或视角来回答

**差的提示词：**
```
"帮我准备一次谈判。"
```

**好的提示词：**
```
"你是我背包制造公司的面料供应商。我正在准备与该供应商谈判，目标是降价10%。请以供应商的身份提供：
1. 三个可能反对降价的理由
2. 针对每个反对意见，从我方角度提出反驳论点
3. 供应商可能提出的两个替代方案（而非直接降价）

然后换回我的角色，提供作为买方应如何最好地推进谈判的建议。"
```

**为什么更好：** 通过角色扮演探索谈判的多个视角，提供更全面的准备。角色扮演还能让 Claude 更好地融入特定视角的细节，提升回答的质量。

## 特定任务技巧

### 内容创作

**1. 明确你的受众**

**差的提示词：**
```
"写点关于网络安全的内容。"
```

**好的提示词：**
```
"我需要为小企业主写一篇关于网络安全最佳实践的博客文章。受众技术水平不高，内容应：
1. 易于理解，尽量避免技术术语
2. 实用，提供可快速实施的可操作建议
3. 有趣且略带幽默感

请提供一篇1000字博客文章的大纲，涵盖这些企业主应采取的5大网络安全实践。"
```

**2. 定义语气和风格**

**差的提示词：**
```
"写一段产品描述。"
```

**好的提示词：**
```
"请帮我为新款人体工学办公椅撰写产品描述。使用专业但吸引人的语气，品牌调性是友好、创新、注重健康。描述应：
1. 突出椅子的关键人体工学特性
2. 解释这些特性如何有益于用户健康和生产力
3. 简要提及使用的可持续材料
4. 以鼓励读者试用的行动号召结尾

目标约200字。"
```

**3. 定义输出结构**

**差的提示词：**
```
"做一个公司业绩的演示文稿。"
```

**好的提示词：**
```
"我需要制作一个Q2业绩的演示文稿，请按以下结构组织：
1. 概览
2. 销售表现
3. 客户获取
4. 挑战
5. Q3展望

每个部分请建议3-4个要点，并推荐一种适合该部分的数据可视化类型（如图表、折线图等）。"
```

---

### 文档总结与问答

1. **明确你想要什么** — 指定摘要的方向、内容类型和输出结构
2. **使用文件名称** — 用文件名称指代附件文档
3. **要求引用来源** — 要求 Claude 在回答中引用文档的具体段落或页码

**差的提示词：**
```
"帮我总结这份报告。"
```

**好的提示词：**
```
"我附上了一份50页的市场研究报告《2023年科技行业趋势》。请提供一个聚焦于AI和机器学习趋势的2段摘要。然后回答以下问题：
1. 今年企业AI应用的前三名是什么？
2. 机器学习如何影响科技行业的职位角色？
3. 报告中提到了哪些AI落地的潜在风险或挑战？

回答时请注明具体章节或页码。"
```

---

### 数据分析与可视化

**差的提示词：**
```
"分析我们的销售数据。"
```

**好的提示词：**
```
"我附上了一份名为《2023年销售数据》的电子表格。请按以下格式分析数据并呈现核心发现：

1. 执行摘要（2-3句话）

2. 关键指标：
   - 各季度总销售额
   - 表现最佳的产品类别
   - 增长最快的地区

3. 趋势：列出3个值得注意的趋势，每项附简要说明

4. 建议：提供3条数据驱动的建议，每条附简要理由

分析完成后，建议三种能有效呈现这些发现的数据可视化方式。"
```

---

### 头脑风暴

**1. 生成多元想法**

**差的提示词：**
```
"给我一些团队建设活动的想法。"
```

**好的提示词：**
```
"我们需要为20人的远程团队策划团队建设活动。请帮我头脑风暴：
1. 提出10个促进协作的线上团建活动
2. 每个活动简要说明如何促进团队协作
3. 标注哪些活动最适合：
   a) 破冰
   b) 提升沟通
   c) 锻炼解决问题的能力
4. 各推荐一个低成本和高端方案"
```

**2. 要求特定格式**

**差的提示词：**
```
"比较几款项目管理软件。"
```

**好的提示词：**
```
"我们正在考虑三款项目管理软件：Asana、Trello 和 Microsoft Project。请用表格格式，按以下标准进行对比：
1. 核心功能
2. 易用性
3. 可扩展性
4. 定价（如有可能请注明具体方案）
5. 集成能力
6. 最适合的场景（如小团队、企业、特定行业）"
```

---

## 排查问题、减少幻觉、提升表现

1. **允许 Claude 承认不确定性** — 告诉它"如果不确定，直接说不知道就好"
2. **拆解复杂任务** — 如果任务过大，拆成小步骤逐一完成
3. **每次对话提供完整上下文** — Claude 不会记住之前的对话，每次需重新提供必要背景

## 综合示例

### 示例1：营销策略制定

**差的提示词：**
```
"帮我制定营销策略。"
```

**好的提示词：**
```
"作为资深营销顾问，请帮我为新型环保智能手机配件制定全面的营销策略。目标受众是注重环保的千禧一代和Z世代消费者。请提供包含以下内容的详细策略：

1. 市场分析：行业趋势、2-3个竞争对手策略、市场规模预测
2. 目标受众画像：理想客户描述及痛点
3. 营销组合：产品亮点、定价策略、渠道选择、5个推广渠道对比及3个创意campaign
4. 内容策略：5个内容主题及建议内容类型
5. KPI与衡量：5个关键指标及推荐工具

请用标题和要点呈现，并在适当处说明推理或举例。最后指出潜在挑战并提供应对策略。"
```

---

### 示例2：财务报告分析

**差的提示词：**
```
"分析这份财务报告。"
```

**好的提示词：**
```
"我附上了公司Q2财务报告《Q2_2023_财务报告.pdf》。请扮演资深CFO，分析报告并准备董事会简报，结构如下：

1. 执行摘要（3-4句话）
2. 财务表现概览：营收（与上季及去年同期对比）、利润率（毛利/净利）、现金流
3. 关键绩效指标（表格格式，附各KPI说明）
4. 业务板块分析：三大板块表现及原因
5. 资产负债表审查：重大变化及关键比率
6. 前瞻性陈述：Q3的3个预测及2-3个战略建议
7. 风险评估：3个潜在财务风险及应对策略
8. 同业比较：与2-3个竞争对手对比，找出优劣势

适当使用图表可视化数据。对任何假设请明确说明。
分析完成后，列出董事会成员可能提出的5个问题及建议回答。
最后，将整个分析浓缩为一段话，供我作为董事会会议开场白使用。"
```
