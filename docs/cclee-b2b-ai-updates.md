# AI 客服 更新日志

## 2026-04-28 功能重写

本次更新从 7 个基础场景扩展至 **18 个完整场景**，涵盖客户实际使用的全部对话路径。

### 新增功能

| 场景 | 说明 |
|------|------|
| SKU 精确查询 | 客户输入 SKU 编码（如 `8586`），AI 返回精确产品和专属价格 |
| 阶梯定价计算 | 客户询问批量价格时，AI 计算指定数量的订单总价 |
| 订单状态/追踪 | 支持按订单号精确追踪，显示处理进度 |
| RFQ 页面引导 | 客户在报价页面时，AI 提供表单填写上下文指导 |
| RFQ 状态查询 | 客户可查询已提交报价请求的处理状态 |
| 多轮对话 | AI 记忆上下文，支持追问、跟进，自然探索产品后发起 RFQ |
| 跨语言支持 | AI 自动检测语言并回复，支持中文、英文等多种语言 |
| Shortcode 嵌入 | 可用 `[cclee_b2b_chat]` 短代码将聊天面板嵌入任意页面 |
| 对话历史持久化 | 登录用户的对话历史存入数据库，换设备后自动恢复 |
| 清除历史 | 用户可手动清空对话记录 |

### 功能优化

- **产品关键词搜索**：扩展返回信息，包含价格、库存状态
- **库存查询**：增加替代产品推荐
- **速率限制**：调整为每 IP 每 60 秒配置，更灵活
- **意图路由**：增加 SKU 查询、RFQ 相关两个独立意图类型
