# AI Agent Changelog

## MVP (2026-03)

<StatusTag type="info">Initial Release</StatusTag>

### Core Features

- **Natural Language Driven**: Conversational interface, zero code barrier
- **SSE Real-time Streaming**: Every step visible in real-time
- **BYOK Support**: Bring your own API Key, data sovereignty
- **Session Memory + Short-term Memory**: Agent remembers context, learns your preferences

### Supported Models

- Claude (Anthropic)
- GPT-4o (OpenAI)
- DeepSeek
- Gemini (Google)

### Known Limitations

- Single Agent only
- No plugin marketplace yet
- No skill sharing yet
- No online support yet
