## Core Features

### <ChartIcon size={20} /> Real-time Data Acquisition

- **Product Information Extraction**: One-click access to product titles, prices, sales, ratings and other core information
- **Price History Tracking**: Automatically record product price trends
- **Stock Monitoring**: Real-time tracking of product stock status

### <SearchIcon size={20} /> Competitor Analysis

- **Competitor Comparison**: Quickly compare various metrics of multiple competitors
- **Hot Sales Rankings**: Automatically generate category best-seller rankings
- **Keyword Analysis**: Analyze competitor title keyword distribution

### <DownloadIcon size={20} /> Data Export

- **Excel Export**: Support one-click export of data to Excel spreadsheets
- **Batch Image Download**: Batch download product main images and detail images
- **Data Visualization**: Charts display sales trends

## FAQ

### Plugin not displaying data?

**Solutions:**
1. Check if network connection is normal
2. Refresh the page and retry
3. Check if there are latest plugin updates
4. Clear browser cache and retry

### Data export failed?

**Possible causes:**
- Browser permissions not enabled
- Too much exported data

**Solutions:**
1. Check plugin download permissions
2. Export data in batches
3. Use CSV format instead of Excel
