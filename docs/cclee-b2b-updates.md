# AI 客服 更新日志

## 2026-04-28 功能重写

本次更新从 7 个基础场景扩展至 **18 个完整场景**，涵盖客户实际使用的全部对话路径。

### 新增功能

| 场景 | 说明 |
|------|------|
| SKU 精确查询 | 客户输入 SKU 编码（如 `8586`），AI 返回精确产品和专属价格 |
| 阶梯定价计算 | 客户询问批量价格时，AI 计算指定数量的订单总价 |
| 订单状态/追踪 | 支持按订单号精确追踪，显示处理进度 |
| RFQ 页面引导 | 客户在报价页面时，AI 提供表单填写上下文指导 |
| RFQ 状态查询 | 客户可查询已提交报价请求的处理状态 |
| 多轮对话 | AI 记忆上下文，支持追问、跟进，自然探索产品后发起 RFQ |
| 跨语言支持 | AI 自动检测语言并回复，支持中文、英文等多种语言 |
| Shortcode 嵌入 | 可用 `[cclee_b2b_chat]` 短代码将聊天面板嵌入任意页面 |
| 对话历史持久化 | 登录用户的对话历史存入数据库，换设备后自动恢复 |
| 清除历史 | 用户可手动清空对话记录 |

### 功能优化

- **产品关键词搜索**：扩展返回信息，包含价格、库存状态
- **库存查询**：增加替代产品推荐
- **速率限制**：调整为每 IP 每 60 秒配置，更灵活
- **意图路由**：增加 SKU 查询、RFQ 相关两个独立意图类型

### 图片更新

本次更新新增 18 张场景截图，已拷贝至 `static/images/docs/cclee-b2b/ai/` 目录。

## 2026-04-29 知识库增强

### 新增功能

| 功能 | 说明 |
|------|------|
| MD 文件上传 | 知识库支持上传 Markdown 文件，现有格式：PDF、TXT、MD |
| 小 KB 全量注入 | 知识库总量 ≤50KB 时，全部文档直接注入 AI 提示词，无需关键词匹配 |
| 跨语言问答 | 客户用中文提问也能匹配英文知识库文档（如"起订量"匹配"MOQ"） |

### 功能优化

- **分块算法升级**：相邻段落贪心合并，单 chunk 从 500 字符提升至 1500 字符，上下文更完整
- **上传失败提示**：文件上传失败时显示具体原因（格式不符、超限、目录不可写等）
- **删除权限检查**：删除文档前验证文件权限，防止静默失败

### 问题修复

- 修复部分服务器环境下知识库目录权限不正确导致文件上传失败的问题

<InfoBox variant="info" title="跨语言问答说明">
当知识库总量超过 50KB 时，系统自动切换回关键词检索模式。关键词检索模式下，中文提问可能无法匹配英文文档内容，建议文档中使用中英双语关键词。
</InfoBox>
