## 常见问题

### Q: 对话没有响应？

1. 检查是否配置了 API Key
2. 检查 API Key 是否有效（未过期、有余额）
3. 检查网络连接
4. 刷新页面重试

### Q: 工具调用卡住？

1. 检查是否有待确认的工具调用（弹出确认框）
2. 允许或拒绝后继续
3. 检查风险确认设置是否过于严格

### Q: 记忆没有保存？

- 长期记忆由 AI 自动提取
- 需要有足够重要的信息才会保存
- 可在记忆页面手动查看和管理

### Q: 如何使用公开技能？

1. 进入 Skills 页面
2. 切换到「**Public Skills**」标签
3. 查看其他用户分享的技能
4. 在 Agent 设置中绑定使用

### Q: 忘记密码？

- 使用登录页面的「忘记密码」
- 检查邮箱重置链接
