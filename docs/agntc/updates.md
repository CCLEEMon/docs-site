# AI Agent 更新日志

## MVP (2026-03)

<StatusTag type="info">首次发布</StatusTag>

### 核心功能

- **自然语言驱动**：对话式交互，零代码门槛
- **SSE 实时流式输出**：每一步执行实时可见
- **BYOK 支持**：自带 API Key，数据自主
- **会话记忆 + 短期记忆**：Agent 记住上下文，越用越懂你

### 支持模型

- Claude (Anthropic)
- GPT-4o (OpenAI)
- DeepSeek
- Gemini (Google)

### 已知限制

- 仅支持单 Agent
- 暂不支持插件市场
- 暂不支持技能分享
- 暂无在线客服支持
