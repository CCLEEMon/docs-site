# 开发者文档

欢迎来到 CCLHUB 文档站的开发者专区。

## 项目概述

本文档站基于 [Docusaurus 3.x](https://docusaurus.io/) 构建，采用 MIT 协议开源。

## 技术栈

- **框架**: Docusaurus 3.x
- **语言**: TypeScript
- **样式**: TailwindCSS
- **图标**: Lucide React

## 快速开始

```bash
git clone <repository-url>
cd docs-site
npm install
npm run start
```

访问 http://localhost:3004
