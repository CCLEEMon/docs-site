# AI 工具场景推荐

> 最后更新：2026-03-18
> 数据来源：CCLHub API 聚合 + 工具导航

## 设计原则

- 按用户场景组织，而非工具类型
- 同一工具可出现在多个场景
- 每个场景按成本/能力排序
- 标注 🔄 表示 CCLHub 已聚合 API
- 每个工具最多 3 个场景标签
