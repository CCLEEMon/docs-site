## FAQ

### Q: No response in chat?

1. Check if API Key is configured
2. Check if API Key is valid (not expired, has balance)
3. Check network connection
4. Refresh page and retry

### Q: Tool call stuck?

1. Check if there's a pending tool call confirmation (popup)
2. Continue after allowing or rejecting
3. Check if risk confirmation settings are too strict

### Q: Memory not saved?

- Long-term memory is automatically extracted by AI
- Needs sufficiently important information to be saved
- Can manually view and manage in memory page

### Q: How to use public skills?

1. Go to Skills page
2. Switch to "**Public Skills**" tab
3. View skills shared by other users
4. Bind and use in Agent settings

### Q: Forgot password?

- Use "Forgot Password" on login page
- Check email for reset link
