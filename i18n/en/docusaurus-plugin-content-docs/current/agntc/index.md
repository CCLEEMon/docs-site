## Core Features

### <MessageCircleIcon size={20} /> Natural Language Driven

- **Chat to Operate**: Describe your needs in plain language, Agent understands and executes
- **Zero Code Barrier**: No technical background required, non-technical business users can use it easily
- **Smart Correction**: Made a mistake? Just correct it in conversation, Agent adjusts in real-time

### <EyeIcon size={20} /> Full Transparency

- **Real-time Progress**: Every step is visible in real-time via SSE streaming
- **Stop Anytime**: Something's off? One-click pause, fully controllable
- **Complete Logs**: All operations are traceable for audit and review

### <KeyIcon size={20} /> BYOK (Bring Your Own Key)

- **Data Sovereignty**: Your API Key, your data, doesn't pass through our servers
- **Cost Transparency**: Direct model calls, no hidden fees, pay for what you use
- **Multi-model Support**: Claude, GPT-4o, DeepSeek, Gemini - your choice

### <ShieldIcon size={20} /> Sandboxed Security

- **Triple Protection**: Code review + runtime sandbox + user authorization
- **Isolated Execution**: Plugins run in isolated environment, cannot access other data
- **Transparent Permissions**: Permissions clearly listed at install, revocable anytime

### <BrainIcon size={20} /> Smart Memory

- **Learns You Better**: Automatically remembers your preferences and habits
- **Three-layer Memory**: Session, short-term, long-term memory, retained as needed
- **One-click Clear**: View and delete memories anytime, data fully under your control
