## 1. 功能概述

### 1.1 智能对话

- **流式输出**：实时显示 AI 响应
- **工具调用**：Agent 可调用外部工具获取信息
- **Tavily 搜索**：联网搜索实时信息
- **技能增强**：绑定技能增强 Agent 能力

### 1.2 三层记忆系统

| 记忆类型 | 存储时间 | 用途 |
|----------|----------|------|
| 会话记忆 | 当前会话 | 临时对话上下文 |
| 短期记忆 | 7 天 | 最近交互记录 |
| 长期记忆 | 30 天 | 重要偏好/背景 |

**长期记忆**由 AI 自动从对话中提取，存储用户偏好、项目背景等重要信息。

## 2. 技能系统

### 2.1 什么是技能？

技能是预定义的 Prompt 模板，可以增强 Agent 的特定能力。

### 2.2 技能类型

技能库页面包含两个标签：
- **My Skills**：你创建的技能
- **Public Skills**：其他用户公开分享的技能，可直接使用

### 2.3 创建技能

1. 进入左侧边栏「**Skills**」页面
2. 点击「**New Skill**」
3. 填写：
   - **名称**：技能名称（必填）
   - **描述**：技能用途说明
   - **System Prompt**：AI 角色定义（必填）
   - **模型覆盖**（可选）：为技能指定独立的 LLM 供应商和模型
   - **公开设置**：开启后其他用户可发现并使用此技能
4. 点击「**Save Skill**」

### 2.4 使用示例模板

如果你是新手，可以使用预置的示例模板：
1. 进入技能库页面
2. 在空列表状态下会显示示例技能
3. 点击示例即可基于模板创建

### 2.5 技能示例

```
名称：代码助手
描述：帮助编写和优化代码
System Prompt: 你是一个专业的编程助手，擅长编写干净、高效的代码...
```

## 3. 记忆管理

### 3.1 进入记忆页面

有两种方式进入 Agent 的记忆管理：
1. **聊天页面**：点击顶栏的「Memory」按钮
2. **设置页面**：在 Agent 设置页点击「Manage Memory」按钮

### 3.2 查看记忆

记忆按三个标签页分类：
- **会话记忆**：当前对话的临时上下文
- **短期记忆**：7天内自动过期
- **长期记忆**：按重要度排序，30天过期

### 3.3 更新记忆

对于**长期记忆**，你可以：
- 调整重要度分数（importance score）
- 添加或修改标签（tags）

### 3.4 清除记忆

- 可按类型清除（会话/短期/长期）
- 可清除所有记忆
- 清除后不可恢复

## 4. 任务历史

### 4.1 聊天页任务历史

任务历史显示在**聊天页面右侧**的侧边栏中，始终可见：

- 显示该 Agent 的最近任务列表
- 包含：
  - 对话摘要
  - 执行状态（进行中/已完成/失败）
  - 时间戳

### 4.2 Dashboard 最近任务

在 Dashboard 页面中部也会显示**所有 Agent 的最近任务**，方便快速了解整体活动情况。

## 5. 最佳实践

### 安全建议

- 为每个 Agent 配置适当的风险等级
- 定期清理不需要的 API Key
- 不要在对话中分享敏感信息

### 效率建议

- 为特定任务创建专用 Agent
- 使用技能系统增强能力
- 合理配置记忆层级

### 成本优化

- 根据任务复杂度选择合适的模型
- 简单任务使用轻量模型（如 GPT-4o-mini、DeepSeek-V3）
- 复杂推理使用高级模型（如 Claude 4、GPT-4o）
