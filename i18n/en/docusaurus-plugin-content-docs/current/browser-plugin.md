## Installation & Usage

### Installation Steps

1. **Download Plugin**
   - Official download: https://cclhub-releases.oss-cn-shenzhen.aliyuncs.com/cclhub-latest.zip

2. **Install Plugin**
   - Chrome: Drag and drop the .crx file to the extensions page
   - Edge: Visit the extensions management page, load the unpacked extension

3. **Register Account**
   - First-time use requires CCLHUB account registration
   - Supports quick phone number registration

### User Guide

**Quick Start:**

1. Open the target e-commerce website (Taobao, Tmall, JD, Pinduoduo, etc.)
2. Browse product pages, the plugin will automatically display the data panel
3. Click the plugin icon to view detailed data and analysis results
4. Use the export function to save data locally

## FAQ

### Plugin not showing data?

**Solutions:**
1. Check if network connection is normal
2. Refresh the page and try again
3. Check if the plugin has the latest version update
4. Clear browser cache and try again

### Data export failed?

**Possible causes:**
- Browser permissions not enabled
- Export data volume too large

**Solutions:**
1. Check plugin download permissions
2. Export data in batches
3. Use CSV format instead of Excel
